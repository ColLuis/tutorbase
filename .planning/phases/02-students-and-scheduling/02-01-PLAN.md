Create `src/lib/utils/time.ts` with two exported functions:

**parseTimeInput(input: string): { hours: number; minutes: number } | null**
- Accepts: "3:45", "14:00", "2:15 pm", "2:15pm", "3", "15:30"
- Regex: `/^(\d{1,2}):?(\d{2})?\s*(am|pm)?$/i`
- Parse hours from match[1], minutes from match[2] ?? '0', period from match[3]
- Apply AM/PM conversion: if period === 'pm' && hours < 12 → hours += 12; if period === 'am' && hours === 12 → hours = 0
- Validate: hours must be 0-23, minutes must be 0-59
- Return null for any input that fails to match or fails validation

**combineDateTime(dateStr: string, timeStr: string, timezone: string): Date | null**
- dateStr format: 'YYYY-MM-DD' (from calendar picker value)
- timeStr: the raw free-text time input (passed through parseTimeInput first)
- If parseTimeInput(timeStr) returns null, return null
- Build local datetime string: `${dateStr}T${String(hours).padStart(2,'0')}:${String(minutes).padStart(2,'0')}:00`
- Convert to UTC using `fromZonedTime(new Date(localDatetime), timezone)` from 'date-fns-tz'
- Return the resulting UTC Date

Import: `import { fromZonedTime } from 'date-fns-tz'`

Both functions must be exported as named exports.
